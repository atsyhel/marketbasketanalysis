# marketbasketanalysis
Market basket analysis using FP-Growth algorithm in PySpark (Databricks).

**Description**  
Project focused on market basket analysis using transactional data.
The goal was to identify frequent itemsets and discover patterns in customer purchasing behavior.

**Tools and technologies**
- PySpark
- Databricks
- FP-Growth algorithm

**Scope of work**
- Data loading and preprocessing
- Transformation of transactional data
- Frequent pattern mining with FP-Growth
- Interpretation of frequent itemsets
